# 강인혁 · 홍은채 모바일 청첩장

강인혁과 홍은채의 결혼을 알리는 모바일 청첩장입니다.

## 예식 안내

- 일시: 2026년 10월 4일 일요일 오후 12시 30분
- 장소: 아펠가모 공덕 라로브홀
- 주소: 서울 마포구 마포대로 92 효성해링턴스퀘어 B동 7층

## 주요 기능

- 웨딩 사진 갤러리, 좌우 스와이프, 확대 보기
- 예식장 위치와 지도 앱별 길찾기
- 축하 연락, 계좌 안내, 카카오페이 송금
- 방명록 작성 및 최신순 페이지 보기
- 카카오톡 공유와 청첩장 주소 복사
